# Car Inspection Platform

## Technical Stack

### Car Profile Service
- **Backend**: Helidon SE (4.1.6) with Java 21
- **Database**: AWS DynamoDB with Enhanced Client
- **API**: RESTful with Helidon WebServer
- **Documentation**: OpenAPI with Helidon OpenAPI integration
- **Utilities**: Lombok, MapStruct for object mapping
- **Testing**: JUnit Jupiter (5.12.2)

### Inspection Service
- **Backend**: Helidon SE (4.1.6) with Java 21
- **Database**: AWS DynamoDB with Enhanced Client
- **Metrics**: Micrometer with Prometheus integration
- **Scheduling**: Helidon Scheduling
- **Documentation**: OpenAPI with Helidon OpenAPI integration
- **Utilities**: Lombok, MapStruct for object mapping
- **Testing**: JUnit Jupiter (5.12.2)
